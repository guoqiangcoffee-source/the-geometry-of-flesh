# 肉身的几何
### 一名外科医生的物理学笔记

### *The Geometry of Flesh*
### *A Surgeon's Notes on Physics*

> **核心主张**：人体不是一堆器官的集合，而是**一组在空间中展开的形式**。血管是分形，肺泡是表面张力的解，心包是闭合曲面，神经是网络，心律是相空间中的轨迹，肿瘤是幂律分布的集团。这些形式是不可见的——它们不在 CT 片上，不在病理切片上，不在任何一本解剖学图谱上——但它们决定了身体如何工作、如何失败、如何愈合、如何死亡。本书试图把这些不可见的形式，一个一个地写下来。

<p align="center"><b>Anatomy = Form + Function</b></p>
<p align="center"><i>形 即 是 功</i></p>

## 关于这本书

我是一名外科医生。

每天的工作是切开身体、看见器官、缝合伤口。在很长一段时间里，我以为自己在和"身体"打交道。直到某一刻我意识到——我面对的不是身体，而是一组**形式**：一种比器官更抽象、比细胞更深层、比基因更先在的东西。

血管为什么是分形的？肺泡为什么不会全部塌掉？心律为什么必须是非线性的？疼痛为什么会"放射"？疾病为什么有"进程"？这些不是生物学问题——它们是几何问题、流体问题、信息问题、尺度问题。它们的答案，不在解剖图谱里，而在数学和物理里。

这本书，是手术之外的笔记。

它不是教科书。也不是科普。它是一组思辨随笔——每一章从一个具体的解剖事实或临床瞬间出发，调用一个具体的数学或物理学概念，展开一段克制而精确的思考。
